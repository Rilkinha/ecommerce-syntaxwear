# SyntaxWear - Tênis e Sneakers Online

Este é o repositório para o site da SyntaxWear, uma loja de e-commerce online especializada em tênis e sneakers.

## Descrição

O site da SyntaxWear é uma página estática criada com HTML e CSS, com um design moderno e responsivo, que se adapta a diferentes tamanhos de tela. Ele apresenta uma página inicial com um banner de herói, seções de categorias de produtos, uma grade de produtos em destaque e um rodapé completo com links de navegação e informações de contato.

## Estrutura de Pastas

A estrutura de pastas do projeto está organizada da seguinte forma:

```
/
├── index.html
├── README.md
└── assets/
    ├── css/
    │   ├── base.css
    │   ├── cartegory-product.css
    │   ├── footer.css
    │   ├── global.css
    │   ├── header.css
    │   ├── hero.css
    │   ├── product-grid.css
    │   ├── reset.css
    │   └── variables.css
    └── img/
        ├── banner/
        ├── favicons/
        ├── icons/
        ├── logo/
        └── products/
```

- **index.html**: O arquivo principal da página inicial.
- **README.md**: Este arquivo.
- **assets/**: Contém todos os recursos do site.
- **assets/css/**: Contém todos os arquivos de estilo CSS.
- **assets/img/**: Contém todas as imagens, logotipos e ícones.

## Tecnologias Utilizadas

- HTML5
- CSS3

## Como Usar

Para visualizar o site, basta abrir o arquivo `index.html` em seu navegador da web.

## Customização

### Conteúdo

O conteúdo do site, como os títulos, textos e links, pode ser editado diretamente no arquivo `index.html`.

### Estilos

Os estilos do site, como cores, fontes e layout, podem ser personalizados editando os arquivos CSS na pasta `assets/css/`.

- **variables.css**: Contém as variáveis CSS para cores e fontes.
- **header.css**: Estilos para o cabeçalho.
- **footer.css**: Estilos para o rodapé.
- **hero.css**: Estilos para a seção de herói.
- **cartegory-product.css**: Estilos para as categorias de produtos.
- **product-grid.css**: Estilos para a grade de produtos.
- **base.css**: Estilos de base para o corpo, botões, etc.
- **global.css**: Estilos globais e de responsividade.
- **reset.css**: Redefine os estilos padrão do navegador.

### Imagens

As imagens podem ser substituídas na pasta `assets/img/`. Certifique-se de que os novos nomes e caminhos de arquivo correspondam aos utilizados no `index.html` e nos arquivos CSS.

## Créditos e Agradecimentos

Este site foi criado como um projeto de exemplo. As imagens e logotipos são apenas para fins de demonstração.
