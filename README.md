# SyntaxWear - Tênis e Sneakers Online

Um site de e-commerce de tênis e sneakers, com um design moderno e focado na experiência do usuário.

## Descrição

O SyntaxWear é um projeto de e-commerce desenvolvido para a venda de tênis e sneakers. O site apresenta um design limpo e moderno, com foco na apresentação dos produtos e na facilidade de navegação. A página inicial é composta por um hero section com um modelo principal, seções de categorias de produtos, e um grid de produtos em destaque.

## Tecnologias Utilizadas

*   **HTML5:** Para a estruturação do conteúdo da página.
*   **CSS3:** Para a estilização e o design do site, utilizando variáveis para um tema consistente.
*   **Design Responsivo:** O site foi desenvolvido para se adaptar a diferentes tamanhos de tela, como desktops, tablets e smartphones.
